# 🖥️ Website Portfolio

This repository contains the website with the portfolio developed in the HTML and CSS training from byron.solutions.
The example file has the original style, created according to the training, and the index file has the personal version, with some personal changes in style and content.

## 🚀 Technologies Used

- **HTML:** Semantic web structuring.
- **CSS:** Modern styling and responsive layout.

## 📁 Project Structure

Below is an overview of how the repository files are organized:

```bash
website-portfolio/
├── assets/
│   ├── css/
│   │   ├── normalize.css
│   │   ├── style-example.css   # reference and example style
│   │   └── style.css           # main portfolio styling
│   └── img/                    # images used in the project
│       ├── cards/
│       ├── examples/
│       ├── about-img.png
│       └── home-img.png
├── index.html                  # Main portfolio landing page
└── example.html                # Example portfolio
```
